# Diş Kliniği Yönetim Sistemi

### Proje Hakkında
Bu proje, BGT 132 dersi kapsamında geliştirilmiş bir masaüstü uygulamasıdır. Diş kliniklerinde hasta kayıtlarını yönetmek amacıyla tasarlanmıştır.

### Proje Yapısı
* **src/**: Uygulamanın Python kaynak kodlarını içerir.
* **docs/**: Gereksinim analizi ve UML diyagramlarını içerir.
* **data/**: Kaydedilen hasta verilerinin tutulduğu klasördür.

### Kullanılan Teknolojiler
* **Dil:** Python
* **Arayüz:** Tkinter
* **Konsept:** Nesne Yönelimli Programlama (OOP)

### Nasıl Çalıştırılır?
1. Python'un yüklü olduğundan emin olun.
2. `src/main.py` dosyasını çalıştırın.
